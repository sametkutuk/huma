#  HUMA - Türkçe Alfabe Öğrenme Uygulaması

Çocukların Türkçe alfabeyi eğlenceli ve interaktif bir şekilde öğrenmelerini sağlayan web uygulaması.

##  Özellikler

-  **29 Türkçe Harf** - Tıklanabilir interaktif kartlar
-  **Sesli Geri Bildirim** - Her harf için açıklama
-  **3 Farklı Tema**: Alfabe, Çiftlik Hayvanları, Ev Eşyaları
-  **Ses Ayarları**: Hız ve ton ayarlanabilir
-  **Özel Açıklamalar** - Kişiselleştirilebilir
-  **Otomatik Kayıt** - Ayarlar tarayıcıda saklanır

##  Kullanım

1. Bir harfe tıklayın
2. Sesli açıklamayı dinleyin
3. Ayarlar'dan özelleştirin

##  Hedef Kitle

3-7 yaş arası çocuklar için tasarlanmıştır.
